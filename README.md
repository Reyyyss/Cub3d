# Cub3d

## Clone

Clone with submodules:

```sh
git clone --recurse-submodules git@github.com:miovu/Cub3d.git
```

If you already cloned without submodules:

```sh
git submodule update --init --recursive
```

## Build

```sh
cd Cub3d
make
```

This builds the `cub3d` binary at the project root.

## Run

```sh
./cub3d
```

## Clean

```sh
make clean
make fclean
```
